# Bingo
Tool to start a python http server in a simple way.

# Installation:
-Manual install
Run:
```bash
chmod +x bingo
sudo apt-get install python3 pip toilet 
sudo mv ./bingo /usr/bin/bingo
```
-Automatic install
Run:
```bash
sudo install.sh
```
and 
```bash
pip install -r requirements.txt
```
-Usage:
```bash
bingo (port)  
```
- only bingo command will open a server on port 8000
